# 🎓 AttendAI - Face Recognition Attendance System

## 📌 Overview
AttendAI is an AI-based attendance system that uses face recognition to automatically mark attendance. It eliminates manual effort and improves accuracy using real-time computer vision.

---

## 🚀 Features
- 🎥 Real-time Face Recognition using OpenCV
- 🧑‍🎓 Automatic Attendance Marking
- 📊 Dashboard with Analytics & Charts
- 🛠 Admin Panel (Add/Delete Users)
- 🔐 Login & Logout System
- 📁 CSV Attendance Storage

---

## 🛠 Tech Stack
- Python (Flask)
- OpenCV (Computer Vision)
- SQLite (Database)
- HTML, CSS, JavaScript (Frontend)
